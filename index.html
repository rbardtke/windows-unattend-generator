Distribution Share ($OEM$ Folders)

About Distribution Shares

You can include additional files in your installation media using the $OEM$ folder structure:

  • $OEM$\$1\ - Files copied to C:\
  • $OEM$\$$\ - Files copied to C:\Windows\
  • $OEM$\$Progs\ - Files copied to C:\Program Files\
  • $OEM$\$Docs\ - Files copied to user profiles

Place the $OEM$ folder in the same directory as your autounattend.xml file on the installation media.
